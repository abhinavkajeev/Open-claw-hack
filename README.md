# Vibe Architect Agent

This repository contains three main parts:

- `agent/`: OpenClaw judging logic and skill metadata.
- `frontend/`: Next.js dashboard for uploads, wallet connection, and vibe certificate display.
- `contracts/`: Solidity reference contracts for identity/registry linkage.
